    	<p>Blender was the first 3D modeling software I used and it sparked my intrest in CAD design. I started by watching YouTube videos on the basics of blender and when I got comfortable with it I tried to start modeling without tutorials. After almost a year I realized that Blender's artistic focused features were fun but wasn't the kind of work I wanted to do with 3D modeling.</p>
	</div>
	<div class="text_column">
		<h2 class="cad_head">Inventor</h2>
    	<p>Autodsek Inventor was recommended to me by my IT instructor when he saw the work I was interested in. Inventor is now the main CAD software I use and have been using it for over a year. The first things I made were tabletop replicas of real robots that compete in the show. As time progressed I started modeling functional things to 3D print that would solve a problem in my everyday life. I also use Inventor to make or prototype parts to fix or build projects. </p>
	</div>
	<div class="text_column">
		<h2 class="cad_head">Aspire</h2>
    	<p>Aspire is software that I use if I need a part made of a material like wood or foam to be cut using CNC. The main thing I use Aspire to do is convert models I made in Inventor to gcode that can be read by the mill. Though I don't use it often, Aspire does have modeling capabilities and can be used in a number of creative and useful ways. </p>
	</div>

let slideIndex = [1,1];
let slideId = ["mySlides1", "mySlides2"]
showSlides(1, 0);
showSlides(1, 1);

function plusSlides(n, no) {
  showSlides(slideIndex[no] += n, no);
}

function showSlides(n, no) {
  let i;
  let x = document.getElementsByClassName(slideId[no]);
  if (n > x.length) {slideIndex[no] = 1}    
  if (n < 1) {slideIndex[no] = x.length}
  for (i = 0; i < x.length; i++) {
     x[i].style.display = "none";  
  }
  x[slideIndex[no]-1].style.display = "block";  
}
</script>
